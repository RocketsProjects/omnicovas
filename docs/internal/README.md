# Internal Documentation

This directory contains internal development notes, planning documents, and AI-assistant workflow files. These are not part of the user-facing product documentation.

## Contents

- **blueprints/** — Architectural blueprints, compliance matrices, and roadmaps used to guide development.
- **dev-guides/** — Phase-specific development guides and task planning documents.
- **ai-workflow/** — AI-assisted code generation prompts, execution plans, and reference documents used during development.
- **archive/** — Superseded versions of blueprints, roadmaps, and earlier planning artifacts (reference only).
